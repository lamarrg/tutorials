# chmod octals - way to change permissions with octal numbers (base 8) 

# octal   Binary  File mode
# 0       000     ---
# 1       001     --x
# 2       010     -w-
# 3       011     -wx
# 4       100     r--
# 5       101     r-x
# 6       110     rw-
# 7       111     rwx

> chmod 755 <file or folder>   (sets file permissions to -rwxr-xr-x)
